### 📁 Projects

- ** [Venus](https://venus.trackalways.cloud/live-map)** — Venus is the customer-facing product brand name for the Fleetpro fleet management platform. It is a comprehensive system designed to track, monitor, and manage fleets of vehicles, devices, and drivers in real-time. The platform features a highly stylized, cross-platform user interface ("Venus OS") and includes an integrated AI assistant ("Venus AI") that allows users to query vehicle statuses and manage operations using natural language.. *(Closed source — ask me about the architecture.)*
- ** [Carpool](link)** — The Carpooling project is a comprehensive system designed for fleet tracking and managing internal ride requests. It uses a modern tech stack consisting of a NestJS backend, a PostgreSQL database with PostGIS for spatial data, Redis for caching, and Traccar Cloud for telematics. The repository is well-structured, containing dedicated codebases for the backend, web, and mobile applications, alongside infrastructure configurations..
- ** [Home Matchmaker](https://realestate-nu-gray.vercel.app/)** — Full-stack student housing app with swipe-to-match discovery and in-app payments (React, Node.js, Supabase).
- ** [AfriFood](https://afrifood.vercel.app/)** — Web and mobile recipe app for traditional African cuisine, backed by a custom relational schema in Supabase.
